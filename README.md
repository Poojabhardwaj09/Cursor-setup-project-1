# Cursor Setup Project

## Tools Installed

* Cursor IDE
* Claude Code Extension
* Codex Extension
* GitHub

## Steps Completed

1. Installed Cursor IDE from official website
2. Installed Claude Code and Codex extensions inside Cursor
3. Created a public repository on GitHub
4. Opened the repository in Cursor
5. Created this README file
6. Committed and pushed the project to GitHub

## Issues Faced & Solutions

* Could not find proper login option for extensions  
  → Tried reinstalling and restarting Cursor

* Repository already existed error while publishing  
  → Resolved by publishing with a new repository name

* Confusion in connecting local folder to GitHub  
  → Used "Publish Branch" option in Cursor


## What I Learned

* Basic GitHub workflow (commit, push)
* How to use an AI-based IDE (Cursor)
* Importance of self-learning and problem-solving


## Reflection
This project helped me understand how to set up a development environment, work with GitHub, and troubleshoot issues independently. I learned the importance of exploring solutions and continuing progress even when facing technical difficulties.
